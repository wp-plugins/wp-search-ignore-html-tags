Updated the native wordpress search to ignore html tags in post and page content.

== Description ==

This plugin updates the native wordpress search function to ignore html tags in post and page content.

== Installation ==

1. Upload 'wp-search-ignote-html-tags' to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
